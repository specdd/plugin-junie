# Contributing

This plugin repository is generated from the SpecDD agent plugins source
repository:

https://github.com/specdd/agent-plugins

Please do not open pull requests that edit generated plugin files directly.
Changes to skills, manifests, shared assets, documentation, or build behavior
should be made in `specdd/agent-plugins` and then rebuilt into this plugin
repository.

For bugs, feature requests, documentation improvements, and implementation
proposals, use:

https://github.com/specdd/agent-plugins/issues

For broader SpecDD framework questions or proposals, use:

https://github.com/specdd/specdd

Before contributing, read the source repository README and the applicable
`.sdd` specs. SpecDD specs are source-adjacent development contracts.

By contributing, you agree that your contributions will be licensed under the
Apache License 2.0.
