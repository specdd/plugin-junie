# Security Policy

This plugin repository is generated from the SpecDD agent plugins source
repository:

https://github.com/specdd/agent-plugins

Please report security issues privately by emailing hello@specdd.ai.

Do not open public GitHub issues for suspected vulnerabilities.

Include enough detail to reproduce or understand the issue, including affected
versions, commands run, relevant files, and any proof of concept if available.

Security fixes should be made in `specdd/agent-plugins` and rebuilt into the
generated plugin repositories. We will review reports and coordinate fixes
before public disclosure where appropriate.
